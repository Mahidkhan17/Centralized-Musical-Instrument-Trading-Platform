# 🎵 Centralized Musical Instrument Trading Platform

## 📌 Introduction
The **Centralized Musical Instrument Trading Platform** is a desktop-based application designed to manage musical instrument shop operations efficiently. It replaces manual record-keeping with a computerized system for better accuracy, speed, and data management.

---

## 🚀 Features
- 🎸 Manage instrument details (name, price, color, stock)
- 🏪 Dealer management system
- 🛒 Sales and purchase tracking
- 📊 Automatic stock updates
- 📈 Sales and purchase reports (weekly, monthly, yearly)
- 🔐 Login authentication for admin

---

## 🖥️ Existing System
In the traditional system:
- All records are maintained manually in registers
- Difficult to manage large inventory
- High risk of data loss or errors
- Time-consuming searching and updating process

---

## ✅ Proposed System
The computerized system:
- Automates data entry and calculations
- Reduces manual errors
- Provides fast access to information
- Improves stock management
- Generates reports efficiently

## 📂 Modules
1. **Instrument Module**
   - Add, update, delete instrument details
2. **Dealer Module**
   - Manage dealer information
3. **Sales Module**
   - Record sales and calculate total price
4. **Stock Module**
   - Automatically updates inventory
5. **Reports Module**
   - Generate sales and stock reports

---

## 📊 Scope
- Reduces paperwork
- Easy stock management
- Quick search functionality
- Secure login system
- Report generation

---

## ⚠️ Limitations
- No online payment integration
- Only handles instrument sales (no repair services)
